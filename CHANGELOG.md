# Changelog

All notable changes to ThoughtCodec should be documented here.

## 0.1.1 - 2026-07-07

- Removed process-only files from the public repository, including logo exploration notes, generated brand references, publication audit notes, local release notes, publishing checklists, template placeholders, and empty fixture folders.
- Kept the public project focused on the reusable deliverables: bilingual README files, bilingual documentation, five skills, examples, catalog metadata, final visual assets, validation scripts, and GitHub community files.
- Updated documentation and validation rules so the cleaned repository structure has no broken local links or required-file checks for removed process artifacts.

## 0.1.0 - 2026-07-07

- Named the project ThoughtCodec / 思维编解码.
- Added bilingual README files.
- Added MIT license.
- Added project visual assets: logo, banner, social preview, architecture diagram, decompression/compression diagram, and agentic loop diagram.
- Added a monochrome logo variant and matching banner and social preview assets.
- Added five bilingual skills:
  - Architecture Scaffolding / 架构级脚手架生成器
  - SOP Instantiation / 标准作业程序实例化引擎
  - Rule And Prompt Optimizer / 规则与提示词迭代器
  - Code Abstraction And Encapsulation / 代码与组件重构器
  - Personal Methodology Engine / 个人方法论引擎
- Added bilingual catalog metadata.
- Added bilingual contribution, conduct, security, issue, and pull request materials.
- Upgraded validation to check bilingual skill paths, catalog entries, visual assets, image links, and secret-like patterns.
