# SOP Instantiation Example / 标准作业程序实例化示例

## Scenario

A user wants to prepare a public GitHub release for a bilingual AI skill library.

用户希望为一个中英双语 AI Skill 库准备 GitHub 公开发布。

## Prompt

```text
Use the SOP Instantiation Skill.

Goal: Prepare the first public GitHub release for ThoughtCodec.
Deadline: This week.
Available materials:
- README files
- Five bilingual skills
- Catalog JSON
- Visual assets
- Validation scripts

Output:
- Milestone plan
- SOP table
- Decision points
- Risk controls
- Final release checklist
```

## Expected Output Shape

- Objective and scope.
- Milestone plan from content review to public release.
- SOP table with inputs, actions, outputs, and checks.
- Decision points such as license, repository visibility, and release timing.
- Risks such as private data exposure, broken links, and missing examples.

## Review Checks

- The plan has clear completion criteria.
- Every milestone has a verification step.
- External publishing actions are separated from local preparation.
- Review notes can be compressed into future rules.
