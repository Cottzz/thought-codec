# Security / 安全说明

Please do not publish secrets, tokens, private prompts, customer data, personal contact details, account screenshots, or proprietary source material in this repository.

请不要在本仓库中发布密钥、token、私人提示词、客户数据、个人联系方式、账号截图或专有来源材料。

## Reporting / 报告方式

If you find sensitive information in the repository, open a minimal issue that says sensitive content exists, but do not repeat the secret in the issue body.

如果你发现仓库中存在敏感信息，请创建一个最小化 Issue，说明存在敏感内容，但不要在 Issue 正文中重复该敏感信息。

After the repository is public, maintainers should remove the content from Git history when necessary and rotate exposed credentials.

仓库公开后，如有必要，维护者应从 Git 历史中移除相关内容，并轮换已暴露凭证。
