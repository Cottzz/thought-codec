# Assets / 视觉资产

This folder contains project-level visual assets.

本目录存放项目级视觉资产。

## Files / 文件

- `logo.svg`: project logo / 项目 Logo
- `logo-mono.svg`: monochrome logo / 单色 Logo
- `banner.svg`: README banner / README 顶部图
- `social-preview.svg`: GitHub social preview image / GitHub 社交预览图
- `social-preview.png`: upload-ready GitHub social preview image / 可直接上传的 GitHub 社交预览图
- `diagrams/architecture.svg`: architecture diagram / 系统架构图
- `diagrams/decompression-compression.svg`: decompression and compression diagram / 解压与压缩流程图
- `diagrams/agentic-loop.svg`: agentic loop diagram / 智能体闭环图

Skill-specific assets should live under the related skill folder.

某个 Skill 专属的图片应放在对应 Skill 文件夹中。
