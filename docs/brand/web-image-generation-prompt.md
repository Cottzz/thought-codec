# Web Image Generation Prompt

Use this prompt in ChatGPT web image generation to explore ThoughtCodec logo directions. The generated images are for visual direction only; final assets must be redrawn as clean SVG.

```text
Create a logo exploration board for an open-source software project named ThoughtCodec.

Project meaning:
ThoughtCodec is a bilingual AI skill library. It turns abstract human intent into executable workflows, then compresses repeated practice back into reusable skills.

Visual direction:
Modern open-source infrastructure logo, engineering-oriented, rational, clean, precise, and trustworthy. Inspired by abstract geometric software logos, not consumer app icons.

Core metaphor:
Compression and decompression as a continuous methodology engine:
- inputs converge inward
- a geometric processing core transforms them
- outputs expand outward
- optional subtle loop or iteration feeling

Style requirements:
- flat vector logo design
- deep navy background
- cool cyan / ice blue primary mark
- no gradients, no glow, no 3D, no shadows
- no photorealistic mockup
- no concrete wall background
- no decorative sparkles
- no brain, gear, robot, chat bubble, or generic code brackets
- no text inside the mark
- do not use the word METHODOX
- do not include a tagline

Board format:
Generate a clean white or dark inspection board with 8 distinct logo mark directions arranged in a grid.
Each direction should be a simple symbol, not a full poster.
Vary the design strategy:
1. geometric processing core
2. compression/decompression arrows
3. abstract infinity loop
4. modular skill network
5. protocol signal mark
6. hexagonal methodology engine
7. negative-space transformation mark
8. wordmark-compatible app icon

Evaluation target:
The best option should work as:
- GitHub repository logo
- README header mark
- favicon at 16px and 32px
- monochrome version
- social preview symbol

Avoid:
overly complex diagrams, literal flowcharts, dense branching trees, fragile thin lines, generic AI sparkle marks, stock logo templates, and text-heavy lockups.
```
