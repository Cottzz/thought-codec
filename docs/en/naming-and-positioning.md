# Naming And Positioning

## Project Name

**ThoughtCodec**

## Chinese Name

**思维编解码**

## Repository Slug

`thought-codec`

## Tagline

Compress experience into skills. Decompress intent into execution.

## One-Sentence Description

ThoughtCodec is a bilingual skill library that turns abstract human intent into executable AI workflows, then compresses repeated practice into reusable agent skills.

## Target Users

- AI power users who want reusable workflows instead of one-off prompts.
- Developers using Codex, Claude, Cursor, or other coding assistants.
- Operators who turn personal methods, SOPs, and review habits into repeatable systems.
- Open-source contributors interested in prompt engineering, agent workflows, and personal knowledge systems.
