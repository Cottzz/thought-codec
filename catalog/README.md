# Catalog / 目录索引

`catalog/skills.json` is the machine-readable bilingual index for published skills.

`catalog/skills.json` 是已发布 Skill 的机器可读双语索引。

Each skill entry should include:

- `id`
- `title` and `titleZh`
- `type`
- `status`
- `models`
- `tags`
- `description` and `descriptionZh`
- `paths.en` and `paths.zh-CN`
- `examples`

每个 Skill 条目应包含：

- `id`
- `title` 与 `titleZh`
- `type`
- `status`
- `models`
- `tags`
- `description` 与 `descriptionZh`
- `paths.en` 与 `paths.zh-CN`
- `examples`
