# Code Of Conduct / 行为准则

ThoughtCodec aims to be a respectful open-source space for turning practical experience into reusable AI skills.

ThoughtCodec 希望成为一个尊重彼此的开源空间，用于将实践经验沉淀为可复用 AI Skill。

Expected behavior:

- Discuss ideas and changes with care.
- Critique work, not people.
- Be clear, specific, and constructive.
- Respect language differences between English and Chinese contributors.
- Avoid harassment, discrimination, personal attacks, and public exposure of private information.

我们期待：

- 认真讨论想法和修改。
- 批评作品，不攻击个人。
- 表达清晰、具体、建设性。
- 尊重中英文贡献者之间的语言差异。
- 避免骚扰、歧视、人身攻击和公开私人信息。

Maintainers may remove comments, issues, pull requests, or contributors that violate these expectations.

维护者可以移除违反以上准则的评论、Issue、Pull Request 或贡献者权限。
