# Roadmap / 路线图

ThoughtCodec is starting as a compact bilingual skill library. The roadmap keeps the first public release focused while leaving room for integrations and community growth.

ThoughtCodec 当前从一个紧凑的中英双语 Skill 库开始。路线图会先保证首个公开版本清晰可用，再逐步扩展集成与社区协作。

## v0.1.0 - First Public Release

- [x] Project identity: ThoughtCodec / 思维编解码.
- [x] Bilingual README and documentation entry points.
- [x] Five initial bilingual skills.
- [x] Machine-readable bilingual catalog.
- [x] Project logo, banner, social preview, and architecture diagrams.
- [x] GitHub issue and pull request templates.
- [x] Structure validation script.
- [x] Practical example for each initial skill.
- [ ] Public GitHub repository created.
- [ ] Initial commit pushed to `main`.
- [ ] GitHub Actions check confirmed.
- [ ] GitHub About, topics, and social preview configured.
- [ ] `v0.1.0` release published.
